- **Dashboard** → **Paneli**
- **Tenant** → **Hapësirëmarrës** (singular), **Hapësirëmarrësit** (plural)
- **Admin/Administrator** → **Menagjues** (for badges and labels)
- **Users** → **Përdoruesit**
- **Roles** → **Rolet**

## Notes

- "Menagjues" is used for admin badges and labels (singular form)
- "Hapësirëmarrës" is the singular form, "Hapësirëmarrësit" is plural
- "Rolet" is used instead of "Rollet" for "Roles"
- "Ka Paguar" is used for "Paid" status
- "Qasur" is used instead of "aksesuar" in some contexts (as per user preference)

## Date Format

Dates are displayed using the browser's locale date format, but labels are translated:
- Created: → Krijuar:
